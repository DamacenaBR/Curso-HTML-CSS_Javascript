Atributos
    autofocus - fixa inicialmente o curso no input automaticamente
    checked- ele faz que o checkbox ou radio inicie checago (selecionado) ou nao
    requered - obriga a preencher os dados no elemento
    maxlength - numero maximo de caracteres
    multiple - trabalho junto com o tag select - podendo selecionar mais de uma opção
    placeholder - texto de dica do que deve fazer no input

    max - o maximo de numero que pode colocar no imput
    min - o minimo de numero que pode colocar no imput
    cols - colunas
    rows - linhas
    size - quantidade - podendo ser utilizado com select - mostrando a quantidade de elementos escolhendo no size 
    step - o valor que vai pular (utilzado no range)
